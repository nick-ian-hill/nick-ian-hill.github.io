Psychoacoustic Lab

Bridging Experimental Psychology, High-Performance Audio, and AI Orchestration.

The Psychoacoustic Lab is a multi-disciplinary project designed to demonstrate how specialized domain knowledge can be encoded into AI systems. It combines a high-performance Web Audio synthesis engine with an AI-driven configuration layer, allowing for the rapid design and execution of rigorous auditory experiments.

Capability 01

Scientific Rigor & Validation

The lab is built on a foundation of rigorous automated testing that extends far beyond basic DSP math. Every aspect of the experimental pipeline is verified to ensure scientific integrity:

  • Adaptive Logic: Unit tests verify that the StaircaseController precisely follows Levitt-style rules (e.g., 2-down 1-up), including correct step-size reductions and threshold calculations.
  • Robot Observers: Automated "observers" with pre-defined psychometric functions simulate hundreds of trials to verify that adaptive procedures converge to the correct theoretical thresholds.
  • Sequence Control: The engine supports hierarchical block randomization and counterbalancing, ensuring that order effects (like learning or fatigue) are controlled across experimental conditions.
  • Temporal Precision: Tests validate sample-accurate timing for Inter-Stimulus Intervals (ISI) and global envelopes, preventing unwanted clicks or transients that could distract the user or provide confounding cues.

Capability 02

AI Collaboration via Skills

While the engine is "dumb," the design process is "smart." By utilizing AI Skills (specialized packages of knowledge and workflows), a general-purpose LLM is transformed into a domain-expert Lab Assistant.

These Skills, such as psychoacoustic-designer, equip the agent with "empirical yardsticks" by grounding its suggestions in seminal literature (e.g., Levitt, 1971; Green, 1988). The agent doesn't just generate JSON; it understands confound control, automatically suggesting frequency and loudness roving to ensure the participant is responding to the intended cue.

Agentic Workflow: The Gemini CLI, powered by specialized MCP tools and Psychoacoustic Skills, helps a researcher design a complex experiment through a collaborative, multi-step dialogue.

Capability 03

Model Context Protocol (MCP)

The optional MCP Server provides the agent with a dedicated toolkit for stimulus math. Instead of the LLM attempting to calculate complex ERB-spaced frequencies or Schroeder phases (which are prone to arithmetic errors), it delegates these tasks to the MCP server. This "Smart Server" architecture ensures that the resulting experiment configurations are mathematically perfect and scientifically sound.

Capability 04

Industrial-Grade Data Integrity

For large-scale research studies, data loss is not an option. The lab includes advanced resilience features designed to protect every trial:

  • Automatic Backups: With autoSave enabled, progress is incrementally saved to local storage, allowing participants to resume exactly where they left off after a crash or accidental exit.
  • Progressive Saving: Real-time lifecycle events allow integrators to stream results to their own servers as each block is completed, ensuring data capture even if a session is never finalized.
  • Seeded Reproducibility: Every session utilizes a master seed, ensuring that randomized sequences remain consistent across recovery events and are fully auditable after the study.
